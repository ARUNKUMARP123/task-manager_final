# Sprint 1 Task Manager

A responsive Task Manager web application built using HTML, CSS, JavaScript, and Bootstrap.

This project was developed for the TIFC2DEVJAVA-SM-TUC-AS2 Sprint 1 Assessment.

## Features

- Responsive Bootstrap Task Form
- Task Form validation
- Task Card layout
- Task List component
- 5 sample task cards
- Dynamic task status change
- Status dropdown options
- Bootstrap responsive UI
- Separate JavaScript validation file

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5

## Functionalities

### Task Form

Users can:

- Add Task Name
- Add Description
- Assign tasks
- Select Due Date
- Choose task status

### Task Validation

Users receive:

- Required field validation
- Meaningful error messages
- Validation on form submission

### Task Cards

Each task card contains:

- Task Name
- Description
- Assigned To
- Due Date
- Status Badge
- Change Status Button

### Dynamic Status Change

Users can update task status dynamically:

↓  
IN PROGRESS  
↓

↓  
DONE  
↓  
TODO
